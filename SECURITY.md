# Security Policy

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please do not report it through public GitHub issues.

Instead, please report it using one of the following methods:

- **Private Advisory:** Use GitHub's built-in private advisory feature by navigating to the Security tab of the repository and reporting a vulnerability.
- **Email:** Contact us directly at rutova2@gmail.com.

We will acknowledge receipt of your vulnerability report within 48 hours and strive to keep you informed about our progress towards a fix and public announcement.

## Supported Versions

Security updates are currently provided for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |
